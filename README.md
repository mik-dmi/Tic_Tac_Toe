# Tic Tac Toe Game (Link for Live Demo: https://mik-dmi.github.io/Tic_Tac_Toe/  )

## Introduction

Welcome to the Tic Tac Toe game! This is a simple web-based implementation of the classic game that you can play in your browser.

## Assignment

This project aimed to demonstrate proficiency in web development using HTML, CSS, and JavaScript. The primary objectives included:

- Setting up the project structure with HTML, CSS, and JavaScript files.
- Implementing the game's logic with a minimal use of global code.
- Organizing code into modules and factories where appropriate.
- Rendering the game board on the webpage.
- Allowing players to add marks to the board by clicking on cells.
- Implementing game logic to prevent illegal moves and check for a win or a tie.
- Enhancing the user interface with player name input, a game start/restart button, and victory messages.
